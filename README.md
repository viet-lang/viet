# viet
The Vietnamese toy programming language

### goals
- Easy to learn, to use and to understand
- Simple syntax, close to Vietnamese
- No semicolon
- Fully unicode support
- Falling in code with Vietnamese language!

### examples

```js
biến lô = "Hello"
biến biểuCảm = " ( *^-^)ρ(*╯^╰)"
in lô + biểuCảm
in 'Cậu ăn cơm chưa?'
```
